# Contributing

See [CONTRIBUTING.md](https://github.com/akz4ol/spec-test-generator-skill/blob/main/CONTRIBUTING.md) for the full contribution guide.

## Quick Start

```bash
# Clone and install
git clone https://github.com/akz4ol/spec-test-generator-skill.git
cd spec-test-generator-skill
pip install -e ".[dev]"

# Run tests
make test

# Run linters
make lint

# Format code
make format
```

## Pull Request Process

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Run tests and linters
5. Submit a pull request

## Code Style

- Use `ruff` for linting and formatting
- Use `mypy` for type checking
- Follow existing code patterns
